## Properties of Four-Vector Transformations

### Invariant Interval

The fundamental invariant in Minkowski spacetime is the spacetime interval

$$ds^2 = dx^2 + dy^2 + dz^2 - c^2dt^2$$

which remains unchanged under Lorentz transformations. This is analogous to the invariance of length in Euclidean space under rotations.

### Four-Vector Dot Product

For any two four-vectors $A = (A_x, A_y, A_z, A_4)$ with $A_4 = i c A_t$, and $B = (B_x, B_y, B_z, B_4)$, the dot product

$$A \cdot B = A_x B_x + A_y B_y + A_z B_z + A_4 B_4 = \mathbf{A} \cdot \mathbf{B} - c^2 A_t B_t$$

is a Lorentz scalar (invariant). This is a powerful tool: constructing Lorentz-invariant quantities from four-vectors.

### Lorentz Transformation Matrix

The Lorentz transformation can be written as a matrix $\mathbf{L}$ such that

$$X' = \mathbf{L} X$$

where $X$ is the four-vector. The transformation matrix satisfies

$$\mathbf{L}^T \mathbf{G} \mathbf{L} = \mathbf{G}$$

where $\mathbf{G}$ is the metric tensor, expressing the invariance of the spacetime interval.

### Applications

The transformation properties of four-vectors allow us to:

1. Transform physical quantities between reference frames without coordinate-by-coordinate calculation
2. Identify invariants that have the same value in all inertial frames
3. Construct covariant equations that automatically satisfy the principle of relativity
