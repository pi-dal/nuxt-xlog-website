## 応用 — 崩壊過程と不変質量

### 粒子崩壊

粒子崩壊 $A \to B + C$ は重要な応用例です。四元運動量保存

$$P_A = P_B + P_C$$

と不変量 $P^2 = m^2 c^2$ を用いることで、基準系を指定することなく崩壊生成物のエネルギーと運動量の関係を導出できます。

### 不変質量

系の不変質量は

$$M^2 c^2 = \left(\sum_i E_i\right)^2 - \left(\sum_i \mathbf{p}_i c\right)^2$$

と定義され、ローレンツ不変量です。重心系では系の静止質量に等しくなります。

### 閾値エネルギー

$A + B \to C + D$ のような反応では、四元ベクトル不変量を用いて粒子生成の閾値エネルギーをエレガントに計算できます。最終粒子が重心系で静止しているときが最小エネルギーとなり

$$E_{threshold} = \frac{(m_C + m_D)^2 c^2 - (m_A + m_B)^2 c^2}{2 m_B}$$

複雑な座標変換を避けて直接正しい結果が得られます。
