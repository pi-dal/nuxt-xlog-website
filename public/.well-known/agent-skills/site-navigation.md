# Site Navigation

Use this skill to navigate the pi-dal website efficiently.

## What it is for

- Discover the latest posts and books
- Jump directly to route-based content pages
- Read the site's API and feed references

## Useful routes

- `/` home
- `/posts` blog index
- `/books` reading notes
- `/chat` chat page
- `/docs/api` API documentation

## Feeds and machine-readable resources

- `/sitemap.xml`
- `/feed.xml`
- `/blog-feed.xml`
- `/books-feed.xml`
- `/.well-known/api-catalog`
- `/openapi/creem-checkout.openapi.json`
