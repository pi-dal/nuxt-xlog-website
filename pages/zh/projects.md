我大部分公开的工作都放在 GitHub 上，但它们并不属于同一种类型。有些是我从头到尾持续构建和维护的项目，有些则是我在既有开源项目中参与修复、文档和站点改进的工作。

这个页面是一个简短索引，用来集中展示最能代表我当前工作方式的项目。

## 我维护的项目

### [django-vditor](https://github.com/pi-dal/django-vditor)

一个把 [Vditor](https://github.com/Vanessa219/vditor) 集成进 Django 管理后台和内容工作流的包。它最初只是一次实用的编辑器接入，后来逐步扩展成一个更完整、更适合生产环境使用的组件库，包含 widgets、上传支持、文档和公开演示站。

- [项目主页](https://django-vditor.pi-dal.com/)
- [源代码](https://github.com/pi-dal/django-vditor)

### [Abraxas](https://github.com/pi-dal/Abraxas)

一个个人 meta-agent / AI-native agent 实验。核心保持得很小，把更多行为能力放在技能、插件、记忆和协作适配层里，让它通过真实对话而不是硬编码规则成长。

- [仓库](https://github.com/pi-dal/Abraxas)
